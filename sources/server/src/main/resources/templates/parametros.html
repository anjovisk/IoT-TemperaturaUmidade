Umidificador:
AUTO: o umidificador é ligado/desligado automaticamente de acordo com a umidade relativa do ar.

o umidificador é desligado se a umidade relativa do ar estiver acima deste valor.
Ar condicionado:
Automático: o ar condicionado é ligado/desligado automaticamente de acordo com os valores de temperatura.

o ar condicionado é ligado se a temperatura estiver acima deste valor.
